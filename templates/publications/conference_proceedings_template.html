Author Manuscript Preparation Instructions

1. Method A: LaTeX2e (download)

LaTeX2e Style Guide

The World Scientific style/template document for LATEX2e, allow authors to prepare manuscripts that can be brought directly into the World Scientific production process for camera-ready titles.

The LATEX2e class file and the sample coded files are located on the website in ZIP format called ws-procs961x669_2e.zip. Once unzipped successfully, you will find the following files:

Filename Purpose
procs-readme961x669 2e.pdf Preliminary guide.
procs-instruction961x669 2e.pdf General instructions for authors & Layout and Font Specification Guide.
ws-procs961x669.cls Main class file.
ws-crop961x669.sty Part of main class file.
ws-procs961x669.bst BIBTEX style file.
ws-pro-sample.bib Sample BIBTEX database.
procs-fig*.eps Sample EPS figures for LATEX users.
procs-fig*.pdf Sample PDF figures for PDFLATEX users.
ws-procs961x669.tex Main TEX file.
ws-procs961x669.pdf Style guide, PDF version of ws-procs961x669.tex.

Apart from the above included support files, this project requires other packages which are by default should available in a good LaTeX2e implementation. If any error message is prompted for a missing package, you can try to download them from a CTAN host.

ws-procs961x669.tex, is the sample file coded in LATEX2e to enable authors to use it as a template and for ready reference. ws-procs961x669.tex uses BIBTEX database to produce the bibliography list.

Getting Help Although a lot of effort has been put in to keep the document style easy to use and provide a concise description of the most common aspects of style, it is of course possible that authors encounter problems. Also authors might have suggestions for improvement. In those cases they can email their comments and suggestions to either of the following: ykoh@wspc.com.sg, rajesh@wspc.com.sg

2. Method B: MS-Word (download)

Introduction

World Scientific has produced a style/template document for MSWord, which will allow authors to prepare manuscripts that can be brought directly into the World Scientific production process for CRC titles of the proceeding size [9.61in x 6.69in]. This will enable more accurate production of page proofs, reducing your need during proofreading to locate typographic mistakes.

Word Template Style Guide

The World Scientific Word templates and sample files are located on the website in a zip format called ws-procs961x669_word.zip. Download and unzip this. Once unzipped successfully, you will find the following files:

procs-readme961x669_word.pdf : preliminary guide
procs-instruction961x669_word : general instructions
ws-procs961x669_word.dotm : style / template file
ws-procs961x669_word.pdf : PDF version of ws-procs961x669_word.dotm

The template can be retained in the current working folder and double clicking it should open a new document based on our styles. Now just overwrite its contents with your text, save and continue editing / styling as required.
